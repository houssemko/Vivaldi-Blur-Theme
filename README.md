# Vivaldi-Blur-Theme
Dark theme with blur for Vivaldi UI.

## Credits & Inspiration

This customization builds upon and adapts layouts, structural selectors, and concepts from the excellent project [Zen-Theme-CSS-for-Vivaldi-Browser by ELGUAPOLIFE](https://github.com/ELGUAPOLIFE/Zen-Theme-CSS-for-Vivaldi-Browser/tree/main). 

Special thanks to the original author for creating a robust foundation for modern Vivaldi setups.

## Core Features

- **Theme-Agnostic Dark Popups**: Built-in variables ensure dropdowns, panels, and modals maintain a clean dark styling with blur effects regardless of active host theme wrapper states (`.theme-dark` / `.theme-light`).
- **Dynamic Titlebar Auto-hide**: Clean workspace integrations that hide standard header layouts seamlessly during UI auto-hide states.
- **Enhanced Address Field Elements**: Subtle translucent design on hover, sleek shadows, and removal of intrusive native borders.
- **Speed Dial Search Enhancements**: Idle blurs transitioning smoothly into deeper backdrop filters on focus/hover for cleaner startpages.
- **Overlays Consistency**: Comprehensive coverage for native overlay states including Quick Commands (`F2`), Ctrl+F Find-in-page box, Clock popups, Privacy dashboards, and `vivaldi:about` page elements.

## File Breakdown

- `custom.css`: The optimized and fully organized stylesheet ready to be applied inside Vivaldi's configuration framework.

## Installation

1. Open Vivaldi and navigate to Settings (`Ctrl+F12` / `Cmd+,`).
2. Go to **Appearance** > **Custom UI Modifications**.
3. Select a folder on your storage drive where you intend to store your custom stylesheets.
4. Drop the included CSS code into a file named `custom.css` inside that folder.
5. Restart your browser or toggle the theme configuration to see the visual layout improvements instantly.
